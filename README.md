# 一人公司财务仪表盘

一人公司，财务一团乱？
**仪表盘帮你看清收支、税务、健康度。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 财务不再"糊涂账"

- **项目盈亏**：按分类分析，Chart.js柱状/饼图
- **税务估算**：3种纳税人类型，自动估算税负
- **健康评分**：综合评分体系，财务健康一目了然
- **趋势追踪**：月度/季度/年度趋势变化

## 怎么用

打开 `index.html` → 填写收支数据 → 查看仪表盘
